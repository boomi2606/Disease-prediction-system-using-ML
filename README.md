# Disease Prediction System using Machine Learning

A machine learning-based web application that predicts the most probable disease based on user-selected symptoms. The project is designed to assist users in obtaining an initial health assessment by analyzing symptom patterns using a trained Random Forest model.

## Features

* Predicts diseases from multiple user-input symptoms.
* User-friendly web interface.
* Machine learning model trained on a symptom-disease dataset.
* Fast and accurate disease prediction.
* Simple and responsive design.
* Easy to extend with additional diseases and symptoms.

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* Scikit-learn
* Pandas
* NumPy

## How It Works

1. The user selects or enters symptoms.
2. The input is preprocessed.
3. The trained Random Forest model analyzes the symptoms.
4. The system predicts the most likely disease.
5. The prediction is displayed through the web interface.

## Project Objective

The objective of this project is to demonstrate the application of machine learning in healthcare by providing an intelligent system capable of predicting diseases based on symptoms. It serves as an educational project that showcases the integration of data science, machine learning, and web development.

> **Note:** This project is developed for educational purposes and should not be considered a substitute for professional medical diagnosis or treatment.
